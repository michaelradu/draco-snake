This is a remake of the original Nokia Snake game in Rust with the Bevy Game Engine.

Check out the documentation below to get started.

## Quick Start Guide

1. Clone the repo

```
git clone https://github.com/michaelradu/draco-snake.git
```

2. Build the project

```
cargo build
```

3. Run the project

```
cargo run
```

4. Have fun!
